Инструкиця для Администратора.
Описание кнопок:

1. Список ключей пользователя - получить перечень ключей конкретного пользователя.
Требуется ввод либо Telegram_ID либо Telegram_login

2. Новый ключ - создается новый VPN Ключ БЕЗ привязки к конкретному пользователю.
Ключ создается неактивный с ограничением траффика в 1кб

3. Удалить ключ - полное удаление VPN ключа. После удаления подключение по этому ключу к VPN сервису будет не возможно.
Требуется ввод ID VPN ключа

4. Привязать ключ к пользователю - привязка пользователя телеграм (необходимо, чтобы пользователь был зарегистрирован в нашей системе) к созданному VPN ключу.
Требуется ввод ID VPN ключа и либо Telegram_ID либо Telegram_login

5. Активация/продление ключа - различные действия над VPN Ключем. Требуется ввод ID VPN ключа для следующих действий:
5.1. Изменить срок действия - изменение срока действия ключа.
Для изменения необходимо указать срок в днях, срок действия будет установлен как текущая дата + количество указанных дней.
если указать 0 - то срок действия будет неограничен.
5.2. Активировать/Деактивировать - изменения статуса активности ключа. Меню меняется динамически в зависимости от настоящего статуса ключа (если статус Активно, то в меню бюдет написано Деактивировать и наоборот)
5.3. Снять лимит трафика/Установить лимит трафика - изменение лимита трафика ключа. Меню меняется динамически в зависимости от лимита (если он есть, то в меню будет Удалить, если нет - Установить)
для Установки лимита трафика необходимо ввести целое число в мегабайтах

6. Отправка сообщений - Требуется выбор как именно отправить сообщение:
6.1. Лично - отправка личного сообщения от имени Бота. Для отправки требуется ввод либо Telegram_ID либо Telegram_login
6.2. Всем - отправка сообщения от имени Бота ВСЕМ зарегистрированным пользователям (за исключением администраторов)
После выбора способа отправки сообщения необходимо ввести сообщение, далее будет предложена проверка введенного сообщения.
Если сообщение корректно - необходимо нажать на кнопку Отправить.
Если в сообщении обнаружена ошибка - необходимо нажать на кнопку Редактировать и повторить ввод сообщения.

7. Памятка администратора - инструкция, с описанием кнопок.

8. Режим пользователя/администратора - переключение между клавиатурами Пользователя и Администратора


АЛГОРИТМ ДЕЙСТВИЯ ДЛЯ АДМИНИСТРАТОРА ПРИ СОЗДАНИИ НОВОГО КЛЮЧА

При отправке клиентом запроса на оформление подписки - всем администраторам придет сообщение от пользователя с указанием его Telegram ID, Telegram login  и сроком, на который он хочет приобрести ключ.
1. Администратор генерирует новый VPN KEY нажав на кнопку п.2 и в ответном сообщении получает ID VPN ключа.
2. Администратор должен привязать ключ к пользователю, для этого он нажимает на кнопку п.4, там указывает ID VPN Ключа и Telegram ID или Telegram login. В ответном сообщении администратор информацию об успешности привязки.
3. После получения оплаты от пользователя - администратор нажимает на кнопку п.5, указывает ID ключа, далее
- Устанавливает срок действия ключа в зависимости от оплаченного периода
- Активирует ключ
- Снимает лимит трафика
